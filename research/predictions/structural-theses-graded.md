## Discipline note — edge attribution must happen AT THESIS-WRITE TIME

Real attribution at thesis-write time captures the ACTUAL conviction source — even if it later turns out to be wrong.

For positions already held (theses written 2026-05-21 to 2026-05-26), I am capturing edge attribution from the EXISTING thesis text, not inventing it. Each stub below cites the thesis file line/section that contains the edge claim.
