## 1. First principles
A technology shock moves the **feasibility frontier** in (capability, unit-economics) space: specific input costs collapse and new primitives appear. Two exploitation modes:
- **Mode A — optimization:** existing business model, cost line reduced. Value largely accrues to incumbents and competes away; frequently isn't even the new technology (Griffin dinner: ML/digitization mislabeled as AI).
- **Mode B — newly viable:** business models whose unit economics were NEGATIVE (or capability impossible) pre-shock, positive post-shock. The "toy" phase is Mode B before demand density arrives.

**The LLM shock's collapsed input:** unit cost of cognition/language work — reading, writing, classifying, judging, conversing (Arora −98%/token vs 3× bills; InferenceX ~60×/yr cost-per-quality, both on file in `sector/end-demand-durability-model.md` §2). **New primitives:** parallel tireless agents; per-user bespoke software; simulated conversation at scale; 24/7 polyglot judgment; near-zero marginal cost per additional "employee" (see `wiki/agent-native-organization.md` lift table).

## 5. Historical base-rate results (landed 2026-07-12 EVE, 1 Opus agent ≈ 28.4k; sources in agent transcript — Statista/MacroTrends/companiesmarketcap/TechCrunch/CB Insights/Menlo T2)

- **Internet cohort (founded 1994-2004, ~$8.31T current value):** at-founding split B ≈ 61% (Google/Meta/PayPal) / A ≈ 35% (Amazon, Netflix). **BUT re-scored by where value was CREATED, Mode B captures ~95%** — the two born-A giants made ~all their value after migrating (Amazon→AWS 2006; Netflix→streaming 2007). - **Mobile+cloud cohort (founded 2007-2015, ~$576B quotable):** **essentially 100% born-B — no pure Mode-A giant exists** (Uber ~$153B, Stripe ~$159B Feb-26 tender, Snowflake ~$92B, Airbnb ~$88B, DoorDash ~$84B). Strongest H1 evidence.
- **The stack-depth insight (the real finding):** the deeper the primitive sits in the stack, the more giants are BORN-B rather than migrated-B. Early web = cheaper distribution channel → first wave looked Mode A, born-B monetization emerged 3-8 years later. Mobile/cloud primitives directly unlocked categories → born-B. Open question for LLMs: cognition-cost collapse is arguably the deepest primitive yet → leans born-B, but copilot-era evidence (below) says the channel phase comes first here too.
- **Graveyard (H3 confirmed, binding on sizing not on search):** ~$5T destroyed 2000-02, Nasdaq −76.8%; Mode-B categories died hardest (Webvan/Pets.com — right idea, too early); 526 "Uber-for-X" listings → Homejoy/Washio/Sprig graveyard. Born-B locates the giants, NOT which bet survives.

Use as "don't reflexively dismiss," never standalone; pair with viability-flip + supply-side survivability screens.
- **Current LLM cycle placement (the headline):** today's largest AI-app revenues are MODE A — coding copilots ~$4bn market (Cursor >$2bn ARR Mar-26, multiple >$1bn ARR), scribes ~$600M, legal ~$650M (CB Insights/Menlo T2) — "same job, faster." **The Mode-B giant of this cycle has NOT been born yet** (or is in its toy phase now). Per the stack-depth pattern + internet analog: expect some winners to start as copilots (channel phase) and MIGRATE — watch migration triggers (assistance→autonomy, per-seat→per-outcome pricing), not only born-B founding.
