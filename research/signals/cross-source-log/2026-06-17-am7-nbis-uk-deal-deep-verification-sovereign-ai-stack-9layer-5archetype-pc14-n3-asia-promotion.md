## PART 2 — SOVEREIGN-AI STACK STRATEGY 9-LAYER + 5-ARCHETYPE TAXONOMY (Subagent B synthesis)

### 9-layer sovereign-AI stack architecture (per Subagent B verification of gov strategy docs)

| Layer | Sovereignty requirement | Substitution rule |
|---|---|---|
| 1. Compute (silicon + power + DC) | Physical infra on domestic soil OR treaty-aligned; power grid not foreign-controlled | Foreign silicon acceptable if in-country + export-controlled (NVIDIA H100/B200 in HUMAIN, Isambard-AI, ABCI 3.0); domestic power required |
| 2. Cloud / orchestration | Tenant isolation, encryption-in-use, data residency, no foreign-gov access | Foreign hyperscaler tolerated if (a) sovereign enclave clauses, (b) neutral-jurisdiction intermediary (OVHcloud, STACKIT, NBIS), OR (c) classified = air-gap only |
| 3. Foundation model | Access to weights, ability to audit, ability to fine-tune; no vendor backdoor | Open-weight (Mistral, Llama, DeepSeek) satisfies most; closed-API only for unclassified commercial; classified = domestic weights |
| 4. Fine-tune / domain model | Gov retains training data + derived weights; no model leakage to vendor | Must be domestic; national-language + domain-specific data creates moat |
| 5. Data | Data residency; national-language corpora; classified-data-handling; GDPR/equivalent | ABSOLUTE requirement — data cannot leave jurisdiction; classified = air-gap + HSM |
| 6. Application / agent | Gov can audit, modify, shut down workflow tools; no vendor runtime dependency | Application layer foreign-built OK IF model + data sovereign (e.g., Mistral to 1M French civil servants) |
| 7. Governance / observability | Kill-switch authority; audit trail; red-team eval; incident response | Contractor-provided OK but gov holds keys; EU AI Act 2026 mandates audit for high-risk |
| 8. Workforce | Domestic AI talent for maintenance, fine-tuning, governance; civil service AI literacy | Cannot be fully outsourced; UNESCO/OPM 2025-26 programs signal gov recognition |
| 9. Standards / regulatory | National legislation determining foreign AI operation terms + audit rights | THE ENFORCEMENT LAYER making all others real; EU AI Act phasing 2026; Korea AI Basic Act Jan 2026; Japan 2026 AI Basic Plan |

**Key insight:** Layer 9 (regulatory) is the LEVERAGE LAYER. EU is the only bloc with binding cross-member enforcement (AI Act) → disproportionate power to set substitution rules for all layers.
