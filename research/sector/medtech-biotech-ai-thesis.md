## TL;DR

The medical-AI pocket is REAL but THIN: 3 GREEN names (HTFL, WAY, Olympus EndoBRAIN segment), all Degiro-accessible. The sector thesis splits into two branches with different economics, different gates, and different falsifiers. **Branch 1 (Application layer)** monetizes AI output directly — the binding gate is the reimbursement code or budget-line replacement. **Branch 2 (Equipment provider)** embeds AI in hardware install bases — the binding gate is the install base itself; AI is feature-uplift and retention. ## Branch 1 — APPLICATION LAYER

**Definition:** AI output IS the product. Monetized per-use, per-study, per-transaction, or subscription. No hardware dependency (runs on existing scanners/EHR/claims infrastructure).

**Economics:** software gross margins; revenue scales with utilization not units shipped; winner-take-most within each reimbursed procedure category.

**Binding gate:** **reimbursement code ownership** (HTFL CPT 75580 canonical) OR **quantified budget-line replacement** (WAY: RCM labor; per T1 anchors in the Phase 3 log — US hospital admin ~199% of direct patient-care costs, ~$40B/yr billing & collections). Without one of these two, application-layer AI is a feature waiting to be absorbed by Epic/EHR vendors or commoditized by foundation models.

## Branch 2 — EQUIPMENT PROVIDER

**Definition:** AI embedded in hardware install bases (scanners, scopes, monitors, sequencers). Monetized via equipment sales + service contracts + software feature-uplift. The AI rides distribution; it rarely carries a standalone price.

**Economics:** hardware gross margins + service annuity; AI improves win-rates and retention more than it creates new revenue lines; almost never discloses discrete AI revenue (the consistent Phase 2 finding across GE/SHL/PHG/Fujifilm/Canon).

**Binding gate:** the **install base itself** + regulatory clearance count. The exception that proves the rule: **Olympus EndoBRAIN earned a standalone reimbursement add-on (厚労省 +60点/¥600 on K721, 2024-06, T1 native-jp)** — the only equipment name where the AI itself is nationally reimbursement-coded. That makes the Olympus EndoBRAIN segment Type-A inside a Branch-2 company.
