# LLMNA

LLM-native investing research operating system. One human (capital + judgment),
Claude sessions (research + execution), this repo (the firm's entire
institutional memory).

- `research/CLAUDE.md` — how the firm works (workflows, conventions, rules)
- `research/INDEX.md` — where everything is (one-hop retrieval index)
- `research/companies/` — one canonical folder per covered name
- `research/portfolio/` — holdings, sizing, risk
- `research/predictions/` — pre-registered, graded forecasts
- `research/signals/` — cross-source logs and triangulation clusters
- `research/sector/` — themes, epochs, bottlenecks
- `research/meta/` — methodology, biases watchlist, lessons, hooks, to-dos

Migrated with full history from `buggatidealership/Health-Calculators` on
2026-07-06. The `main` branch is the live workspace.
